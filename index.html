3. Different font sizes

Absolute font sizes

These font sizes never change once assigned and they are independent of size of parental element. However, they can be used to aid for using relative font sizes. Some of the most commonly used absolute sizing are:

  • {16px} I love Web Development
  • {16pt} I love Web Development


px -> 1/96 inch

pt -> 1/72 inch


Relative font sizes

These font sizes are used to have relative font sizes. The size depends on the parental element. Most common of them are

  • {1rem} I love Web Development
  • {1vh} I love Web Development
  • {1vw} I love Web Development
  • {1em} I love Web Development


  • rem => This sets the font size relative to the document root. e.g we set the font size at root level to 10px (absolute), and in child elements we give 0.5rem, that will be equivalent to 5px
  • vw => It says the font should take 1% of the available width. It can be used with images whose sizes changes depending upon available width and height, so that it doesn't exceed the viewport
  • vh => It says the font should take 1% of the available height. It can be used with images whose sizes changes depending upon available width and height, so that it doesn't exceed the viewport
  • em => It allows setting the font size relative to it's parent. The parent can have absolute sizing and child can have relative sizing
